How to contribute

  1. Fork the repository to your own GitHub account.
  2. Create a new branch for your change (for example, add/your-name).
  3. Edit the file data/contributors.ndjson and add one new line following the example shown in that file.
  4. Commit your change and open a Pull Request.
  5. Once merged, your profile will appear here automatically in a minute or two.

Full instructions are in the README if you need more detail.
